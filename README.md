## Data Governance Charter for Indian Life Insurance Provider

This repository contains a comprehensive Data Governance Charter PowerPoint presentation tailored for an Indian Life Insurance Provider.

### Contents

1. **Data_Governance_Charter_Indian_Life_Insurance.pptx** - A 25-slide PowerPoint presentation covering:
   - Executive Summary
   - Vision and Mission
   - Data Governance Framework
   - Organizational Structure (Data Governance Council, Owners, Stewards, Custodians)
   - Eight Subject Areas with identified owners and stewards:
     - Policy Administration Data
     - Claims Management Data
     - Customer/Policyholder Data
     - Agent/Distribution Data
     - Actuarial and Risk Data
     - Financial and Accounting Data
     - Regulatory and Compliance Data
     - Underwriting Data
   - Roles and Responsibilities
   - Data Governance Policies and Standards
   - Data Quality Management Framework
   - Data Security and Privacy (IRDAI & DPDP Act compliance)
   - Implementation Roadmap
   - Metrics and KPIs
   - Benefits and Value Proposition
   - Regulatory References

2. **create_data_governance_charter.py** - Python script used to generate the PowerPoint presentation

### How to Generate the Presentation

The presentation has already been generated, but if you need to regenerate it:

```bash
# Install required dependency
pip install python-pptx

# Run the generator script
python3 create_data_governance_charter.py
```

### Key Features

- **Comprehensive Coverage**: All major subject areas relevant to life insurance are covered
- **Clear Accountability**: Data Owners and Data Stewards are identified for each subject area
- **Regulatory Compliance**: Aligned with IRDAI regulations and DPDP Act 2023
- **Actionable Roadmap**: Phased implementation plan with clear timelines
- **Measurable Outcomes**: Specific KPIs and metrics for tracking governance effectiveness

### Compliance Framework

The charter addresses key Indian regulations including:
- IRDAI (Insurance Regulatory and Development Authority of India) regulations
- Digital Personal Data Protection Act (DPDP) 2023
- Prevention of Money Laundering Act (PMLA)
- Information Technology Act, 2000
- Industry standards (ISO 27001, ISO 8000)
